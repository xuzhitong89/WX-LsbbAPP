### 律师帮帮法律咨询：
一个咨询付费的律师平台，功能涉及到 图片上传、地理定位、下拉刷新、微信支付等等...
### 目录结构：
- style — 存放独立wxcss文件，可import引入
- image — 存放项目图片
- pages — 存放项目页面相关文件
- fonts —存放字体的ico
- map — 存放地图定位的文件
- template — 存放模板文件
- utils — 存放utils文件，可require引入
